## Fonts (drop your TTFs here)

Put your custom `.ttf` files in this folder and then update the `@font-face` blocks at the top of `styles.css`.

Example:

```css
@font-face {
  font-family: "IV Display";
  src: url("./fonts/YourDisplay.ttf") format("truetype");
  font-weight: 400 900;
  font-style: normal;
  font-display: swap;
}
```
